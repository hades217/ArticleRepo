What is User Experience (UX) ?

User Experience is ....

Everything the user experiences

UX is not just a design, it is everything between the user and the product or service. This includes every tangible and intangible item.

user experience
noun

the overall experience of a person using a product such as a website or computer application, especially in terms of how easy or pleasing it is to use.

"if a website degrades the user experience too much, people will simply stay away"

Some UX issues you may not have considered

The Save Button and the Generation Gap

Research has shown there is a generation gap with saving files:

  • Older generations expect there to be a delay and an animation to show content is saving
  • Where as the millennial generation just expect it always save, that it is always in the cloud

The Save Button

In some instances, online software will show a saving message for a prolonged period of time so the user is comfortable that their file was saved.

This small fake animation made users more comfortable that their work had been saved.

Sometimes existing patterns don't make sense

Like the save icon, a fair significant of younger uses have never experienced or seen a real floppy disk.

Some Generation Z children were shown a floppy disk and assumed it was a 3D printed version of the save icon.

Many designers have tried to create a new save icon but have failed due to popularity of the current icon.

Differences of user experience between generations

Millennials will generally expect things to flow and work without having to read labels, and help text.

Previous generations will take care to read the copy and ensure that they have all the instructions they need before they commence with the form.

Great tangible example of this behaviour is with electronic devices, Blu Ray Player, Playstation etc

User Experience effects with design

The design of a product or service, whether it is funky or professional can have an impact on the user's experience.

A food outlet can have a funky website, and reflect the company's culture.

Whereas, a product / service like a bank needs to be professionally designed, clean and easy to use.
